## Overview

The Authed Platform provides a user-friendly web interface for managing your agents and provider settings. To use Authed, you must first register as a provider through our platform at [getauthed.dev](https://getauthed.dev).

## Getting Started

### 1. Sign in to Platform

Start by signing in to our platform:

  <img src="/images/platform/signin_to_platform.png" alt="Sign in to Platform" />
</Frame>

### 2. Register as a Provider

Complete your provider registration by adding additional information:

<Frame>
  <img src="/images/platform/register_provider.png" alt="Register as a Provider" />
</Frame>

### 3. Save Your Provider Credentials

After registration, you'll receive your provider credentials. Make sure to save these securely:

<Frame>
  <img src="/images/platform/save_provider_credentials.png" alt="Save Provider Credentials" />

### 4. Access Dashboard

Once registered, you'll have access to your provider dashboard where you can manage your agents:

### Creating a New Agent

From your dashboard, you can easily create new agents:

<Frame>
  <img src="/images/platform/agent_create.png" alt="Create a New Agent" />

### Agent Credentials

After creating an agent, you'll receive its credentials. Make sure to save these securely:

<Frame>
  <img src="/images/platform/agent_secret.png" alt="Agent Credentials" />

### Agent Actions

Each agent has its own dedicated section where you can:
- View authentication logs
- Manage permissions
- Monitor agent activity
- Configure agent settings
